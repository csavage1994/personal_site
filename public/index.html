Executive Weekly Report - 5thColumn

Problem

One issue that is bound to come up when dealing with network traffic is the sheer quantity of data. While digging in can provide useful insight for engineers, it is very difficult to get the full picture without significant knowledge. The solution was to create a weekly emailed report, which was also available as a standalone site.

Tech

  • React.js - Front end framework which provides helpful tools for creating interactive and modular user interfaces

  • Node.js - Server side JavaScript; Used specifically to programatically fetch and email reports

  • Cron - Time-based job scheduler used to ensure emails were sent out on time every week

Solution

  • Increased value and confidence for executives by providing a high-level view of their network health

  • Flexible configuration made it trivial to add or remove clients, tweak email times, and recipients

  • Designed report layout and implemented a fully responsive, print-friendly site
